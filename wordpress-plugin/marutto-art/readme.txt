marutto.art の無料イラスト素材（商用利用可）を、投稿エディタから直接検索・挿入できます。

== Description ==

**Marutto Art** は、[marutto.art](https://marutto.art/) の無料イラスト素材をWordPressのエディタから直接検索・挿入できるプラグインです。

= 主な機能 =

* **Gutenberg（ブロックエディタ）対応** — サイドバーパネルから素材を検索し、画像ブロックとして挿入
* **Classic Editor 対応** — メディアボタンからポップアップを開いて素材を挿入
* **キーワード検索** — タイトル・説明・キーワードを横断して素材を検索
* **ページネーション** — 大量の素材を快適にブラウズ
* **登録不要・APIキー不要** — そのままご利用いただけます

= 素材について =

* すべての素材は商用利用可能です
* ブログ・SNS・YouTubeサムネイル・印刷物等に使用できます
* 素材の著作権は marutto.art に帰属します
* 詳細は[利用規約](https://marutto.art/terms-of-use/)をご確認ください

= API =

本プラグインは [marutto.art 公開API](https://marutto.art/api/docs/) を利用しています。
レートリミット：1,000リクエスト / 日（IPアドレス単位）

== Installation ==

1. `marutto-art` フォルダを `/wp-content/plugins/` にアップロードします
2. WordPress 管理画面の「プラグイン」メニューからプラグインを有効化します
3. 投稿・固定ページの編集画面を開きます
4. **Gutenberg**: 右上の「marutto.art 素材挿入」アイコン（画像マーク）をクリック
5. **Classic Editor**: エディタ上部の「marutto.art から挿入」ボタンをクリック

== Frequently Asked Questions ==

= 無料で使えますか？ =

はい、APIキーや登録は一切不要です。1日1,000リクエストまで無料でご利用いただけます。

= 商用利用はできますか？ =

はい、すべての素材は商用利用可能です。詳細は[利用規約](https://marutto.art/terms-of-use/)をご確認ください。

= クレジット表記は必要ですか？ =

任意ですが、「marutto.art」へのリンクを掲載いただけると大変助かります。Gutenbergで挿入した際はキャプションにリンクが自動挿入されます。
